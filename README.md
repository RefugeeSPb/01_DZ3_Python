# Скрипт парсинга книг

Этот репозиторий представляет собой скрипт для парсинга данных о книгах с сайта `books.toscrape.com`. 
Скрипт собирает информацию о книгах, включая название, цену, рейтинг, количество в наличии, описание и дополнительные характеристики.

## Описание

Скрипт использует библиотеку `BeautifulSoup` для парсинга HTML и `requests` для отправки HTTP-запросов. Данные собираются с нескольких страниц сайта и сохраняются в файл `books_data.txt` в формате JSON.

## Установка

Для использования скрипта вам понадобятся следующие зависимости:

- `requests`
- `beautifulsoup4`
- `pytest` (для тестирования)

Вы можете установить их с помощью `pip`:

```bash
pip install requests beautifulsoup4 pytest
```

Также подготовлен файл для установки библиотек:
```bash
pip install -r requirements.txt
```

## Использование

1. **Запуск парсинга**:

   Скрипт можно запустить из командной строки:

   python scraper.py

   Это запустит парсинг и сохранит данные в файл `books_data.txt`.

2. **Тестирование**:

   Для запуска тестов используйте `pytest`:

   ```bash
   pytest tests/test_scraper.py
   ```

   Это запустит тесты для функций `get_book_data` и `scrape_books`.

Проект включает тесты для функций `get_book_data` и `scrape_books`. Тесты проверяют, что функции возвращают корректные данные и что данные сохраняются в файл.
